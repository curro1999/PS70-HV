/Final Project Idea
Introduction
This project aims to scale and customize the existing PioReactor. The idea revolves around taking advantage of the existing software of the company to incorporate my own sensors, personalize with code on the UI, and build the prototype with PS70 technologies.
Why is this relevant?
You like wine? Beer? Well, what makes boring grape juice a tasty wine is fermentation, done by m.i.c.r.o.b.e.s. I work in gastronomy, my focus is on making delicious foods, and one tool I use is selecting different microbes, dropping them together in a cocktail shaker and BANG! Now, in order to do this, I kinda need to control them. The PioReactor is sick, but it only works with single cultures and doesn’t allow other sensors like the ones I use. So hell yeah! Let's build my own! And one of the coolest things is that mixed-cultures are actually quite a hot topic right now in microbiology, but sadly there isn’t much tech about it...
